新增的多媒体标签

1.音频标签

  • 属性 值 描述
  • autoplay autoplay 视频就绪自动播放
  • controls controls 向用户显示播放控件
  • loop loop 是否循环播放
  • src url 视频url地址

2.视频标签:

视频video -常见属性:

  • 属性 值 描述
  • autoplay autoplay 视频就绪自动播放
  • controls controls 向用户显示播放控件
  • width pixels 设置播放器宽度
  • height pixels 设置播放器高度
  • loop loop 是否循环播放
  • preload auto(预先加载视频) none(不应加载图片) ...
  • src url 视频url地址
  • poster imgurl 加载等待的画面图片
  • muted muted 静音播放
  • 注：谷歌浏览器需要添加muted来解决自动播放的问题S

新增的input 类型

type="email" 限制用户输入必须为Email

type="url" 限制用户输入必须为url

type="date" 限制用户输入必须为date

type="time" 限制用户输入必须为time

type="month" 限制用户输入必须为month

type="number" 限制用户输入必须为number

type="tel" 限制用户输入必须为tel

type="search" 限制用户输入必须为搜索框

type="color" 生成一个颜色表单

新增的表单属性

  • 属性 值 说明
  • required required 表单内容不能为空,必填
  • placeholder 提示文本 表单的提示信息，存在默认值不显示
  • autofocus autofocus 自动聚焦属性，页面加载完毕自动聚焦到指定表单
  • autocomplete off/on 默认on
  • multiple multiple 可以多选文件提示
